# Fine-tuning Tools

Utilities for preparing training data and fine-tuning models on EatVid-Bench.

## Files

### generate_cot.py
Generates Chain-of-Thought (CoT) reasoning chains from video annotations.

### prepare_train_data.py
Prepares training data in LLaMA-Factory format with CoT reasoning.

### evaluate.py
Evaluation script for fine-tuned models (requires LLaMA-Factory).
